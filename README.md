kc-android
==========

knowledge camp 的 android 客户端

## 安装相关组件
### 首先要安装ActiveAndroid
```
git clone https://github.com/pardom/ActiveAndroid
cd ActiveAndroid
mvn clean install -Dmaven.test.skip=true
```

### 安装KCAuthenticator
```
git clone https://github.com/mindpin/KCAuthenticator
cd KCAuthenticator
mvn clean install
```
